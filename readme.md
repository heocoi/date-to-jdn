# date-to-jdn

> Converting Gregorian calendar date to Julian Day Number (JDN).

## Install

```sh
npm install date-to-jdn
```

## Usage

```js
import dateToJdn from "date-to-jdn";

dateToJdn(new Date('2024-02-09'));
//=> 2460370
```

## API

### dateToJdn(date)

#### date

Type: `object`

A `Date` object.
